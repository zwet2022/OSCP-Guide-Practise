## Shell stabilizeren

### Overzicht

Shell stabilizeren, zodat dit niet zomaar wegvalt of gek doet.

#### Commando's

```
python3 -c 'import pty;pty.spawn("/bin/bash")'
```

```
stty raw -echo; fg
```

```
export TERM=xterm
```
